# Spec Brief：HLS-05 命名 / HLS-08 KEYFORMAT / BT magnet UDP tracker 过滤

## 范围
1. **HLS-05（最小）**：VOD 产物默认文件名若以 `.m3u8`/`.m3u` 结尾，改为 `.ts`（与 `video/mp2t` 一致）
2. **HLS-08（最小）**：`KEYFORMAT` 缺省或 `identity` 才允许 AES-128；其它 KEYFORMAT 在 parse/probe 明确拒绝
3. **BT privacy（deferred 收口）**：SOCKS 启用时，magnet URI 内嵌 `tr=udp://` 在 `add_magnet` 前剥离，与 session 全局 tracker 过滤一致

## 非目标
- MAP/fMP4/多音轨完整产品化
- private torrent metadata 阶段 DHT 隔离（依赖 librqbit 设计，仅文档诚实化）
- QUIC prior knowledge 启用（需 unstable 构建策略）
