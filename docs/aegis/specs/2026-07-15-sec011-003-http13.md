# Spec Brief：SEC-011 备份导入预算 · SEC-003 OS 确认 · HTTP-13 range final-host

## 范围
1. **SEC-011**：`import_backup` 限制备份文件大小（64 MiB）、任务数（默认 10000），超限拒绝且不改仓库
2. **SEC-003**：`request_confirmation` 在签发 token 前弹出 **OS 原生** `tauri-plugin-dialog` 确认框；恶意 JS 无法在无用户手势下静默拿 token
3. **HTTP-13 深化**：`HttpClient` 记录最近一次响应 final host；`Protocol::last_resolved_host`；引擎在分片 acquire 前刷新 `metadata.resolved_host`，使 CDN 归属进入 per-host 许可

## 非目标
- 中途已持有的 permit 重绑定（仅影响后续 acquire）
- piece-hash 与 snapshot 字节级交叉（BT-17 深度）
- 备份路径 symlink reparse 句柄化
