# Spec Brief：HTTP-13/14/15 · BT-17 · BT-18 pread

## 范围
1. **HTTP-14**：`HttpClient::download_full` 对无 Content-Length 的 body 逐 chunk 累加上限，禁止无界 `.bytes()`
2. **HTTP-15**：`execute_full_download_once` 在 known `expected_size` 下写前拒绝 `pos+chunk > expected`
3. **HTTP-13（最小）**：probe 记录最终 host；`DownloadTask::request_host` 优先用 `metadata.resolved_host` 做 pool.acquire
4. **BT-17（最小）**：`protocol_managed_storage` 时丢弃 snapshot completed/partial 跳过（由 piece truth / FileStream 负责完成语义）
5. **BT-18 补完**：`pread_exact` 循环读满

## 非目标
- 下载中途二次 redirect 动态换 permit
- private torrent metadata DHT（BT-11）
- piece hash 与 snapshot 深度交叉验证
