# Spec Brief：HTTP-09 / HTTP-16 / HTTP-17

## 范围
1. **HTTP-09**：`execute_full_download` 增加与分片路径同构的 `max_retries` + 可重试错误 backoff；每次 attempt 开始将存储截断到 0（或 expected 0 写入起点），避免半写污染
2. **HTTP-16**：`ConnectionPool::acquire` 先 host 后 global，消除跨 host HOL
3. **HTTP-17**：HTTP 连接超时 **唯一 owner = `DownloadConfig.connect_timeout_secs`**；`PoolConfig`/`ConnectionConfig` 的同名字段不传入 HttpClient（文档 + 测试锁定）

## 非目标
- HTTP-15 Client registry（架构更大）
- HLS segment 级 retry（HLS-06，另刀）
