# Spec Brief：BT-19 io_strategy · A-09 effective proxy

## 范围
1. **BT-19**：`TachyonStorageFactory::open_storages_from_metadata` 按 `io_strategy` 打开后端（Standard/WinAligned/Iocp/IoUring，平台不可用时回退 Standard），禁止忽略字段硬编码 TokioFile。
2. **A-09**：`BtSession` 保存构建时解析的 `effective_socks_proxy`；`proxy_coverage_status` 基于 effective 而非仅显式 `MagnetConfig.socks_proxy_url`；报告附带来源与脱敏 endpoint。
3. 前端 MagnetTab 优先展示后端 `get_bt_proxy_coverage` 运行时事实；draft 预测仅作未应用提示。
