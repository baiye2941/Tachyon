# Spec Brief：SEC-009 剪贴板 capability + sniffer 凭据最小化 · SEC-012 UPnP 默认关

## 范围
1. **SEC-009a**：从主窗口 capability 移除 `clipboard-manager:allow-read-text`（后端 `AppHandle.clipboard()` 不依赖 JS 权限）
2. **SEC-009b**：嗅探 IPC/事件列表不再返回 `downloadUrl`；新增 `create_task_from_sniffer` 按资源 id 在后端取原始 URL 建任务
3. **SEC-012**：`MagnetConfig.enable_upnp` 默认 **false**（缩小 quick-xml 局域网攻击面）；用户可显式开启

## 非目标
- openat2 路径竞态（SEC-010）
- BT-17 piece-hash 深度交叉
- Release 签名供应链（SEC-013）
