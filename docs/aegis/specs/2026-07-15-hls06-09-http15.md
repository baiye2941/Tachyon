# Spec Brief：HLS-06 / HLS-09 / HTTP-15

## 范围
1. **HLS-06（最小）**：segment/key `get_bytes` 对可重试错误做有限次退避重试（默认 3，对齐 DownloadConfig 语义的常量）
2. **HLS-09**：`get_text` 返回 final URL；`fetch_media_playlist` 用 final URL 作相对 URI 基址
3. **HTTP-15（最小）**：按连接身份键共享 `HttpClient`/`reqwest::Client`（UA/proxy/timeouts/http2/headers），engine 构造走 registry

## 非目标
- live HLS、MAP、fMP4、多音轨
- HLS segment-level resume checkpoint
- 全局 ConnectionPool 改名（文档已澄清）
